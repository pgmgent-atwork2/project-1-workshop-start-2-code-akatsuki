Volg de stappenplan

  • 1. Visual Studio installeren

    Eerst moet je de tool van Microsoft Visual Studio Code downloaden. Dit is een gratis tool die je kan helpen bij het schrijven van code. Je kan het downloaden via deze link: Visual Studio Code Zo kunnen we beginnen met het coderen

  • 2. Maken van de project folder

    Nu we onze tool hebben om het spel te maken, moeten we eerst een projectmap aanmaken. Het creëren van een specifieke map voor het project biedt duidelijkheid over hoe je verschillende benodigdheden kunt organiseren. Natuurlijk zal deze map ook de code bevatten die we later zullen schrijven.

  • 3. Maken van de Index html

    De index.html is het fundament van een webpagina. Het bevat de basisstructuur en inhoud van de site, inclusief tekst, afbeeldingen en koppelingen. Het is het startpunt waarop verdere ontwikkeling wordt gebouwd, en het definieert de lay-out en organisatie van de pagina met behulp van HTML-elementen.

    Fun fact: "index.html" is de standaardnaam voor het hoofdbestand van een website, waardoor webontwikkelaars en webserverconfiguraties gemakkelijk de startpagina van een site kunnen herkennen en laden.

  • 4.Het maken en koppelen van een CSS- en JS-bestand

    In dezelfde map zullen we zowel de CSS- als JavaScript-bestanden aanmaken en deze koppelen. Het is van essentieel belang om CSS te maken en toe te passen om stijl aan je website toe te voegen, zoals het instellen van een achtergrondkleur of het kiezen van een lettertype.

    In dit geval zullen we CSS gebruiken om de achtergrondkleur toe te voegen en om het speelkader te positioneren. Het JavaScript-gedeelte zorgt voor de logica achter het spel, van hoe het personage springt tot wanneer het personage wordt geraakt.
